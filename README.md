# mail-delivery
Microservice for mail delivery


STACK: 
      
      docker
      
      docker-compose
      
      kafka
      
      zookeeper
      
      aiokafka(for work with kafka in python)
      
      fastapi
      
      celery
      
      rabbitMQ
      
      flower
      
      postgres

HOW TO START:
1) docker-compose up

2) for start producer write in console: python core/producer.py

3) for start consumer wrire in console: python core/consumer.py
